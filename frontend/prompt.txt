# Redesign Plan for NexGen Schedule

## Overview
The goal is to completely redesign the NexGen Schedule web app to achieve a world-class, visually stunning UI that incorporates 3D animations and a modern aesthetic. This will involve implementing a glassmorphism design philosophy, using subtle animations, and ensuring a responsive layout.

## Brand & Identity
- **App Name:** NexGen Schedule
- **Tagline:** "Your time, perfectly orchestrated."
- **Logo:** 3D animated calendar icon with shimmer effect
- **Color Palette:**
  - Primary: Deep Slate (#0F172A)
  - Violet Accent (#7C3AED)
  - Secondary: Electric Indigo (#4F46E5)
  - Surface: Off-white (#F8FAFC) and pure white (#FFFFFF)
  - Text: Near-black (#0F172A) and cool gray (#64748B)
  - Success: Emerald (#10B981)
  - Warning: Amber (#F59E0B)
  - Error: Rose (#F43F5E)
- **Fonts:** "Inter" for body, "Cal Sans" or "Clash Display" for headings

## Global Design Rules
- **Animations:** Implement Framer Motion for page transitions, staggered animations, and hover effects.
- **Navbar:** Design a floating glassmorphism navbar with a sticky effect.
- **Cards & Buttons:** Redesign cards with glassmorphism effects and implement two button variants.
- **Inputs:** Style input fields with floating labels and custom focus effects.
- **Background:** Incorporate subtle animated mesh gradients.

## Page-by-Page Redesign
1. **Landing Page:** Hero section with role cards and animated background.
2. **User Login:** Centered glassmorphism card with animated logo and form fields.
3. **User Dashboard:** Redesigned booking cards with enhanced UI elements.
4. **User Availability:** New design for host list with animated elements.
5. **Host Dashboard:** Stats and recent bookings with improved layout.
6. **Admin Dashboard:** Stats cards and user management with custom styling.

## Technical Requirements
- Framework: React + Vite
- Animation Library: Framer Motion
- CSS: Tailwind CSS
- Fully responsive with dark mode support.

## Deliverables
Complete updated component code for each page, preserving all existing logic while enhancing the visual design.
